HTML

나의 목표는 나만의 웹 사이트를 만드는 것이다. 처음으로 배운 것은 tag이다. 태그를 공부하려면 빈도수 통계를 기반으로 공부하여라. 태그를 의미에 맞게 쓰는 것은 매우 중요하다. 이유는 다음과 같이 들 수 있다.
검색 엔진에 노출이 잘 되게 할 수 있다.
시각 장애를 가진 사람이라도 웹 페이지의 내용을 제대로 알 수 있게 해 준다.
img 태그에 src와 width라는 두 개의 attribute를 사용해보았다. 속성은 태그의 제목만으로는 부족한 정보를 제공하기 위해 쓰인다. 속성의 순서는 상관없다.

두 번째 문단 시작한다. css로 문단 간 거리 조절할 수 있다.
다음으로 배울 것은 링크이다.
링크입니다
이 a라는 태그는 anchor 즉 닻을 의미하며 링크 기능을 뜻한다. 웹 페이지와 웹 페이지를 서로 연결한다는 의미에서 굉장히 중요한 태그이다. target="_blank"는 링크를 클릭했을 때 새창에서 페이지가 열리게 하는 속성이다.

지금까지 웹 페이지를 만드는 법과 그 웹 페이지 사이에 길(링크)을 놓는 법을 배웠다. 여러 웹 페이지를 엮어서 웹 사이트를 만들어봅시다!
